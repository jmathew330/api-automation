# E-Commerce QA API Test Automation

Automated API testing suite for a RESTful e-commerce platform using Postman and JavaScript.

---

## Project Overview

This project contains comprehensive API tests for an e-commerce platform (Fake Store API), designed to validate core functionalities such as products, carts, and users. The test suite verifies response status codes, response times, data structures, field presence, data types, JSON schema, and data integrity.

---

## Tools & Technologies

- **Postman** — API testing and automation platform  
- **JavaScript** — Test scripting language in Postman  
- **Fake Store API** — RESTful e-commerce API under test  
- **Git** — Version control  

---

## Features

- Automated test suites for endpoints including `/products`, `/carts`, and `/users`  
- 50+ JavaScript tests covering:  
  - Status codes and response times  
  - Required fields and data types  
  - JSON schema validation  
  - Data consistency checks with request-response comparisons  
- Dynamic tests using collection and environment variables
